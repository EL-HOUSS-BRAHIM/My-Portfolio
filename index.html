Log Parsing

It reads data from standard input (stdin) and computes various metrics like total file size and the count of HTTP status codes. It utilizes concepts such as file I/O, signal handling, data processing, regular expressions, and exception handling.

Star Wars API

In this JavaScript project, I built a script to fetch and display characters from the Star Wars API based on the movie ID. It leverages HTTP requests, APIs, and asynchronous programming concepts like promises and async/await to handle external API calls. Command-line arguments are used to specify the movie ID, and the script prints character names in the same order as listed in the API response.

This team project involved developing a full-stack web application called "Budgetify," which helps users manage their finances by tracking income and expenses. The project used both front-end (HTML, CSS, JavaScript) and back-end (Node.js, Express, MongoDB) technologies. Monty Interpreter

This project involves creating an interpreter for Monty ByteCodes files, a scripting language that relies on a unique stack. The goal is to implement various opcodes including push, pall, pint, pop, and others to manipulate the stack effectively.
