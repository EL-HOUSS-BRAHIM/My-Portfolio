## üîç How to Check Rankings Properly

### Method 1: Manual Incognito Search
1. Open **Incognito/Private window** (Ctrl+Shift+N in Chrome)
2. Go to google.com
3. Search exact query
4. Count position (1-100)
5. Record result

**Why Incognito?**: Prevents personalized results based on your history

### Method 2: Google Search Console (Most Accurate)
1. Go to: search.google.com/search-console
2. Select your property (brahim-elhouss.me)
3. Go to: Performance ‚Üí Search Results
4. Filter by specific query
5. See average position over time

### Method 3: Free Ranking Tools
- **SerpWatcher** (free trial)
- **Google SERP Checker** (free)
- **Ahrefs SERP Checker** (free)
- **SEMrush** (free limited)

## üí° Pro Tips

1. **Don't Obsess Daily**: Rankings fluctuate. Check weekly, not daily.

2. **Focus on Trends**: One bad week doesn't matter. Look at month-over-month trends.

3. **Long-Tail First**: You'll rank for "Brahim Elhouss portfolio" before "Brahim Elhouss" alone - that's normal!

4. **Celebrate Small Wins**: First page 3 ranking? Celebrate! It's progress.

5. **Document Everything**: When you see a jump, note what you did that week.

6. **Be Patient**: SEO takes 3-6 months. Consistent effort wins.

7. **Test Different Locations**: Rankings vary by location. Check from Morocco and US.
